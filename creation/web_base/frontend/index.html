GlideinWMS VO Frontend Monitoring

The GlideinWMS VO Frontend provides monitoring for users and administrators. Select the monitoring webpage below.

  • Browse RRDs

    Historical Frontend and Frontend groups statistics plotted over time

  • Group Matrix

    Historical statistics for each group along with the comparison mode

  • Historical Status

    Historical usage of the Frontend as seen by the Glideinwms Factory

  • Current Status

    Current status of the Frontend i.e. running/idle jobs and glideins
